## Scoring Guide (for Facilitator)

For each statement:
- **Odd-numbered** items (1, 3, 5, 7, 9): subtract 1 from the user's response.
- **Even-numbered** items (2, 4, 6, 8, 10): subtract the user's response from 5.

**SUS Score = Σ (adjusted scores) × 2.5**

| Score Range | Interpretation |
|-------------|----------------|
| 0–25 | Unacceptable |
| 25–50 | Poor |
| 50–70 | OK |
| 70–85 | Good |
| 85–100 | Excellent |
